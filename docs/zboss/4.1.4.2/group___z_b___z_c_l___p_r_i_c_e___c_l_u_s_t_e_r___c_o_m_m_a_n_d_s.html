#define ZB_ZCL_PRICE_DEFAULT_PRICE_CONTROL_VALUE   0x01
 Controls PriceAck policy for outgoing PublishPrice packets. SE devices can enable this field (refer zb_zcl_price_publish_price_payload_t::price_control) if they want to receive PriceAcknowledgement command as response to PublishPrice command, but this behavior is valid only for SE version >= 1.1. SE 1.2 is used, so this field is set to 1 by default.
