enum zb_zcl_mi_app_domain_hvac_e

Values for 'Application Domain HVAC' type of Multistate Input cluster.

See also
ZCL spec 3.14.11.19.7.1
Note
This application domain is specific for Heating, Ventilation, Air Conditioning (HVAC) type of applications and the enumeration lists possible application usage states.
