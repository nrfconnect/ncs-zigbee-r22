◆ zb_ncp_custom_request_cb_t

typedef zb_uint16_t(* zb_ncp_custom_request_cb_t) (zb_uint8_t param)

The custom request callback.

Parameters
param- the buffer with request data from host and tsn (zb_uint8_t) as a parameter. It's freed by the stack afterwards.
Returns
The length of a response returned by zb_ncp_custom_response if it is called within callback. NCP_RET_LATER if zb_ncp_custom_response is called later.
Note
If returned length is equal to 0, the response is generated and sent automatically.
